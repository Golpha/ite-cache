save()

save(\Psr\Cache\CacheItemInterface  $item) : boolean

Saves an item to the cache engine

Adds an item to the saved container, adding the expire time to it. This method does not actual save the item to the caching system and the class ancesstors must implement it. The concrete cache adapters may not caches the item object itself but may use only its value.

Parameters
